# DataStructures-class
# Data Structures and Algorithms in Python

## Folders
- Stack : پیاده‌سازی کلاس Stack
- Queue : پیاده‌سازی Queue و Circular Queue  
- Linked_list : لیست پیوندی یک‌طرفه و دو طرفه و توابع مربوط به درج و حذف نودها  
- Tree : توابع مربوط به درخت دودویی (شمارش گره‌ها، برگ‌ها، جمع داده‌ها، جستجو و …)  
- Recursive_function : توابع بازگشتی مانند Fibonacci، ضرب و تقسیم  
- Graph : الگوریتم‌های BFS و DFS و Selection Sort و Bubble Sort
